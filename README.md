# CosmeticManager

Cosmetic box spawn control (chance, rarity weights, extra spawns) and optional unlock-all.

R.E.P.O. BepInEx mod. Host-side only.

## Install

Requires [BepInEx 5.x](https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/).

Place `CosmeticManager.dll` in `BepInEx/plugins/`.

## Configuration

Edit `BepInEx/config/maxenterme.CosmeticManager.cfg` after first launch.

## Build

```
dotnet build -c Release
```
